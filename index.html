Você está prestes a aprender CSS Selectors! Os seletores são como você escolhe em qual elemento aplicar estilos.

Anexo 1 - Uma regra CSS

p { 
   margem-inferior: 12px; 
}

Aqui, o "p" é o seletor (seleciona todos os elementos <p>) e aplica o estilo margin-bottom.

Seletor de enésimo tipo com fórmula

:enésimo-do-tipo(An+B)

A fórmula n-ésimo-do-tipo seleciona cada n-ésimo elemento, iniciando a contagem em uma instância específica desse elemento.

Exemplos

span:nth-of-type(6n+2) seleciona cada 6ª instância de umextensão, começando (e incluindo) a segunda instância.
